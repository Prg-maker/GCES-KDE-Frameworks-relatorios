# 📦 Gestão da Configuração e Evolução de Software - KDE Frameworks

> Este repositório reúne a documentação desenvolvida durante a disciplina **Gestão da Configuração e Evolução de Software (GCES)** da Universidade de Brasília (UnB).

---

## 🎯 Objetivo do Projeto

Nosso objetivo é estudar e aplicar práticas de **gestão de configuração, controle de versões, evolução de software e colaboração em projetos reais de código aberto.**

O projeto escolhido para contribuição foi o **[KDE Frameworks](https://develop.kde.org/products/frameworks/)**.

---

## 🧱 Sobre o KDE Frameworks

> *"Uma coleção de bibliotecas e frameworks de software prontamente disponíveis para quaisquer pilhas de software ou aplicações baseadas em Qt em vários sistemas operacionais."*

O KDE Frameworks oferece soluções para funcionalidades frequentemente necessárias, como:

- ✅ Integração com hardware
- ✅ Suporte a formatos de arquivo
- ✅ Elementos de controle gráfico adicionais
- ✅ Funções de plotagem
- ✅ Verificação ortográfica

A coleção serve como **base tecnológica** para o ambiente **KDE Plasma** e os **Aplicativos do KDE**, sendo distribuída sob a **Licença Pública Geral Menor GNU (LGPL)**.
